# Oracle du Labyrinthe des Cycles — WebUI Streamlit

## Installation

```bash
pip install streamlit
```

## Lancer l'application

```bash
streamlit run oracle_streamlit.py
```

L’interface s’ouvrira dans ton navigateur (par défaut sur http://localhost:8501).

Tu peux éditer `oracle_streamlit.py` pour modifier :
- les cartes,
- les cycles,
- les textes,
- ou la structure des pages.
